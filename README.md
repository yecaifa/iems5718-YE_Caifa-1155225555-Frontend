# IEMS5718 Front End

基于 Vue 3 和 Vite 构建的前端，主要用于 IEMS5718 课程开发。

## 🔗 部署地址
[访问网站](https://yecaifa.github.io/iems5718-YE_Caifa-1155225555-Frontend/)

## 📦 技术栈
- Vue 3 (`<script setup>` 语法)
- Vite (快速构建)
- Vue Router (前端路由管理)

## 🚀 本地运行
```sh
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产环境
npm run build
